# Mind Reader 3000 🤫

一款玩家與電腦互相猜數字的小遊戲，分為兩個模式：
- 玩家猜電腦的數字
- 電腦猜玩家的數字（最多猜 6 次）

## 👾 遊戲檔案

- `index.html`：首頁，選擇進入哪種遊戲模式
- `player.html` & `js/playerScript.js`：玩家猜數字的前端與邏輯
- `boss.html` & `js/bossScript.js`：電腦猜數字的前端與邏輯

## 💡 技術使用

- 原生 HTML/CSS/JavaScript
- 基本 DOM 操作與事件監聽
- 錯誤防呆、簡單動畫效果

## 📦 使用方式

1. 打開 `index.html`
2. 點選遊戲模式即可開始
